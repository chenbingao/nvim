# nvim config

## Must be configured

First of all, create a Lua file named "config.lua" in the current directory, it contains the following content:

```lua
return {
  colorscheme = "everforest",
  background = "light",
  python3_host_prog = "/usr/bin/python3",
  font = "LXGWWenKaiMono Nerd Font",
  font_size = 18,
}
```
